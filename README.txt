Developed a comprehensive approach for classifying Bangla news articles using traditional machine learning, deep learning, and hybrid models, including a novel stacking classifier combining BiLSTM and SVM. 
Achieved 94% accuracy on a dataset of 118,404 articles, demonstrating the effectiveness of ensemble methods in Bangla natural language processing. Conducted detailed analysis using confusion matrices, ROC curves, 
and error metrics to evaluate model performance and contribute to advancements in low-resource language classification.
